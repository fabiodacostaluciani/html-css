Anotações CAP 08

Negrito

Para colocar um texto em negrito use o <strong> para semantica ou o <b> para forma!

Italico

Para colocar um texto em itálico use o <en> para semantica (enfase) ou <i> para forma!

Marca texto

Para colocar um texto marcado por uma cor use o <mark>

Sublinhado

Para colocar um texto sublinhado use o <u>

Letras maiores e menores

Para colocar uma palavra maior ultilize <big> (obsoleta) e para menor ultiize <small>!

Texto riscado/deletado

Para ultilizar o texto riscado use o <del>!

texto inserido/sublinhado

Para ulltilizar o texto inserido use <ins>!

Elevado/Sobrescrito

Para ter um texto elevado use <sup>!

Abaixo ou menor

Para usar o texto abaixo H2O. Use o <sub>!

Codigo fonte

Use o <code> O texto ficará assim!
